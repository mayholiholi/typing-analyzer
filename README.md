# Typing Analyzer

https://mayholiholi.github.io/typing-analyzer/

タイピングを計測してmacOSのキーリピート設定を最適化するWebツール。

## 機能

- **単打テスト**: 意図しないキーリピートが発生していないか確認
- **長押しテスト**: リピート開始までの時間とリピート速度を計測
- **連打テスト**: タイピング速度を計測

3つのテスト結果から、あなたのタイピングに最適な`KeyRepeat`と`InitialKeyRepeat`の値を提案します。

## 使い方

1. 上記リンクからツールを開く
2. 3つのテストを実行
3. 「結果を分析」をクリック
4. 表示されたコマンドをターミナルで実行
5. ログアウト→ログインで設定を反映

## 背景

macOSのキーリピート設定が速すぎると、1回キーを押しただけで「あああああ」のように複数文字が入力されてしまうことがあります。このツールは、あなたのタイピングパターンを計測して最適な設定値を提案します。
